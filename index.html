  1. Start up Minecraft.
  2. Make sure you are logged in!
  3. Select "Multiplayer".
  4. If it gives you a warning dialog, click "Proceed".
  5. In the servers menu, click "Add Server".
  6. Server Name can be anything you want it to be.
  7. Server Address will be mc.wwasl.com.
  8. Hit Done, then the server will show up in the menu.
  9. Hover over the server icon (should be a picture of some houses) and click the big triangle pointing to the right.
